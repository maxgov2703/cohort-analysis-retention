# Cohort Analysis of Customer Retention

## Project Goal

This project analyzes customer retention using cohort analysis based on transaction data from an online retail dataset.

The goal is to understand how customer behavior changes over time after their first purchase.

---

## Dataset

Online Retail Dataset (UK-based e-commerce store).

The dataset contains:

- InvoiceNo
- StockCode
- Description
- Quantity
- InvoiceDate
- UnitPrice
- CustomerID
- Country

Total rows: ~540,000 transactions.

---

## Analysis Steps

1. Data cleaning
2. Revenue calculation
3. Cohort identification
4. Cohort index calculation
5. Retention matrix creation
6. Heatmap visualization
7. Revenue cohort analysis

---

## Key Metrics

- Customer Retention Rate
- Average Order Value
- Cohort Revenue

Example results:

- Month 2 retention ≈ 20%
- Month 3 retention ≈ 22%
- Month 6 retention ≈ 25%

---

## Visualization

The project builds a cohort heatmap showing customer retention over time.
